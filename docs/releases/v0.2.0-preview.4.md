# ZSUI v0.2.0-preview.4

## 中文

本预览版将完整组件库改为同屏中英双语，并修正中文文本在原生控件中的宽度估算。

- 五个组件页面的导航、标题、说明、按钮、状态反馈、弹层与组件目录均提供中文和英文。
- 组件目录中的 48 个组件家族使用中文名称和稳定的英文 API 标识。
- 虚拟列表首屏展示真实双语数据，不再显示空白占位行。
- 框架统一修正 Unicode 文本宽度估算，覆盖基础控件、BreadcrumbBar、TabView、InfoBar、TeachingTip、Toast、ContentDialog、Tooltip 与 Workbench；中文不再被当作半宽字符。
- BreadcrumbBar 在空间充足时不再错误省略中文标签。
- DataGrid 调整双语列宽，避免标题和类型值被截断。
- Windows 发布附件仅保留可直接运行的组件库 EXE；预览图片直接展示在更新说明中。

`component-gallery-demo` 仍为显式可选 profile；默认构建仍只包含 `window`、`button`
和 `label`。WebView 仍不在 ZSUI 产品边界内。

## English

This preview makes the complete component Gallery bilingual on screen and
corrects native width estimation for Chinese text.

- Navigation, headings, descriptions, controls, status messages, overlays and
  catalog content are available in Chinese and English across all five pages.
- All 48 catalog families pair a Chinese name with the stable English API id.
- The initial virtual-list viewport now shows real bilingual rows instead of
  empty placeholders.
- Framework-wide Unicode width estimation now covers base controls,
  BreadcrumbBar, TabView, InfoBar, TeachingTip, Toast, ContentDialog, Tooltip
  and Workbench; Chinese is no longer treated as half-width text.
- BreadcrumbBar avoids false ellipsis when enough space is available.
- DataGrid column allocation keeps bilingual headings and type values visible.
- The Windows release has one directly runnable Gallery EXE asset; screenshots
  are embedded below.

The complete Gallery remains opt-in through `component-gallery-demo`. Default
builds remain limited to `window`, `button` and `label`. WebView remains outside
the ZSUI product boundary.
