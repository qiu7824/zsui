# ZSUI v0.2.0-preview.3

This prerelease corrects the Windows typography pipeline and the semantic text
hierarchy used by the optional component Gallery.

## Typography

- `TextRole` now exposes the complete Microsoft Fluent type ramp: caption
  12/16, body 14/20, body-large 18/24, subtitle 20/28, title 28/36,
  title-large 40/52 and display 68/92.
- Heading roles default to semibold 600; body and caption roles remain regular
  400.
- `styled_text` carries a `SemanticTextStyle` through the View tree and gives
  each label the matching semantic line box. Plain `text` remains the body-role
  convenience constructor.
- The Windows renderer selects installed Segoe UI Variable Small, Text and
  Display optical families and falls back to Segoe UI on older systems.
- Win32 `HFONT` sizes now use the current per-window DPI. The owned native edit
  host uses the same 14-DIP body size and font fallback.
- AppKit and GTK4 consume the shared line-height contract through native
  paragraph/Pango layout while retaining their native system font families.

## Component Gallery

- Page headings, descriptions, navigation metadata, card headings and status
  text now use explicit semantic roles instead of identical body text.
- NumberBox inline step buttons keep the official horizontal placement but use
  semantic Chevron icons and the standard control border instead of text
  triangle glyphs. Icon bounds scale with DPI and retain the existing typed hit
  targets.
- Windows icon-font glyphs use grayscale antialiasing to avoid colored
  ClearType fringes at 10–12 DIP.

The Gallery remains behind `component-gallery-demo`; no widget family or native
backend is added to the default feature set. WebView remains outside the ZSUI
product boundary.

## Platform evidence

- Windows: all five release Gallery pages pass native window, screenshot and
  typed-input smoke checks.
- macOS: the AppKit renderer and all widget sources cross-compile.
- Linux: the GTK4 renderer and all widget sources cross-compile.
- Physical AppKit/GTK4 screenshot and interaction evidence remains pending and
  is not reported as complete.
